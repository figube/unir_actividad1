Que es CSS3

 
            Una definición más formal de CSS sería la siguiente que aparece en Wikipedia:
            CSS es un lenguaje de diseño gráfico que permite definir y crear la presentación de un 
            documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer 
            el diseño visual de los documentos web e interfaces de usuario escritas en HTML.

Año de lanzamiento de las versiones CSS

  1. CSS1: 1996

  2. CSS2: 1998
